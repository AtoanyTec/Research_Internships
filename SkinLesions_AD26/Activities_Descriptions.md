# Research Internship  
## Skin Lesion Classification using Convolutional Neural Networks

---

## 📅 Internship Period
- **Start date:** August 10, 2026  
- **End date:** December 4, 2026  

---

## 🎯 General Objective
To develop a complete artificial intelligence system capable of classifying skin lesions from images, integrating a convolutional neural network model with a functional application (front-end and backend) deployed in the cloud.

---

## ✅ Specific Objectives
- Analyze the state of the art in skin lesion classification using CNNs.
- Understand and preprocess a specialized dataset (e.g., ISIC).
- Implement, train, and evaluate deep learning models (ResNet, AlexNet, ConvNeXt, among others).
- Optimize model performance through experimentation.
- Design and implement a backend for model inference (API).
- Develop a front-end for user interaction.
- Integrate the full system (model + backend + frontend).
- Deploy the solution in the cloud.
- Communicate technical results in a clear and structured manner.
